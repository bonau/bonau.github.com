--- layout: post name: opera-1010-qt4-build-version-for-ubuntu title: Opera 10.10 QT4 build version for Ubuntu date: 2009-11-30 21:32:00.000000000 +08:00 categories: - qt4 - opera permalink: /2009/11/opera-1010-qt4-build-version-for-ubuntu.html --- It seems like it only has QT3 version for Opera official site.
However, there's an official QT4 build indeed.

When downloading Opera,
click cancel and change "qt3" in the URL to "qt4",
or in general you may browse the directory where the file lays on, in order to get compressed files instead of packages.

For example,
Opera 10.10 for debian/Ubuntu builds in QT4 (AMD64):
http://snapshot.opera.com/unix/rc-4742/x86_64-linux/opera_10.10.4742.gcc4.qt4_amd64.deb

The directory:
http://snapshot.opera.com/unix/rc-4742/
